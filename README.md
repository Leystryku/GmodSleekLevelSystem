# GmodSleekLevelSystem
Sleek Level System - one of my addons from ~2013, very barebones imo.

- MySQL support
- Integration for TTT and some other gamemodes
- Deathrun support
- Prophunt support
- Zombiesurvival support
- Sandbox support
- Automatic display of level for some gamemodes
